# Web VRC with Voice Chat & Global SNS

## セットアップ

```bash
npm install
npm start
ブラウザで http://localhost:3000/?room=lobby を開く。
```

# Render デプロイ
Build Command: npm install

Start Command: npm start

Web Service としてデプロイ

---

これをそのまま GitHub に push → Render に Web Service としてつなげれば、  
「ルーム付き3D空間＋音声チャット＋全ルーム共通SNS」がオンラインで動く。

もし「SNSをDBに保存したい」「アバターをVRMにしたい」とかあれば、そこからさらに攻めよう。
